***作品概要***
宇宙空間を舞台にしたシューティングアクションゲームです。
３種類の技を駆使して敵をすべて殲滅してください。

***操作方法***
WASDキー			: 移動
Qキー				: 下降
Qキー２度押し		: 急降下
Eキー				: 上昇
Eキー２度押し		: 急上昇

Hキー				: ターゲット切り替え

Jキー				: 射撃
Kキー				: 近接攻撃
Lキー				: ホーミングミサイル掃射

近接攻撃中にKキー	: 連続攻撃
